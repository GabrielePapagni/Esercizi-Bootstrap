1. A cosa serve bootstrap?

Bootstrap è un framework di sviluppo web gratuito e open source.
È progettato per facilitare lo sviluppo web di siti web responsive e mobile-first.
Fornisce CSS oppure JS già pronti per poter sviluppare il sito web più velocemente.

2. Cosa devi importare per creare una pagina web con bootstrap?

Devo importare questa riga di codice scritta qui sotto:
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">

3. Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

CDN è una rete che fornisce diversi tipi di contenuti.
Uno dei vantaggi è la maggiore velocità di caricamento.

4. A cosa serve questo attributo del tag meta?
content="width=device-width, initial-scale=1

L'attributo content="width=device-width, initial-scale=1" del tag <meta>
è usato per controllare il layout delle pagine web sui dispositivi mobili.

5. Che differenza c’è tra Container e Container Fluid?
(prova il container fluid creando una pagina chiamata BTS_1.1.1.html)

Il Container ha larghezze fisse per vari breakpoint,
mentre il Container Fluid occupa sempre il 100% della larghezza disponibile.
